**Instagram Post Generator**

## Overview
This project automates the generation of Instagram-ready posts from property news using RSS feeds and AI tools.

## Workflow
1. Fetch property news via RSS
2. Generate Instagram caption using OpenAI
3. Generate AI image
4. Upload image to Google Drive
5. Store caption and image link in Google Sheets

## Tools Used
- n8n
- OpenAI (Text & Image)
- Google Drive API
- Google Sheets API
